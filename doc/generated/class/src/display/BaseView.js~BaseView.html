    BaseView

    Extends:

    EventTarget → BaseView

    Direct Subclass:

    ContainerView, SpriteAnimator

    BaseView is the base class for organizing the DOM. It extends EventTarget to allow event based communication between Views and their parents or other parts of the application that also inherit from EventTarget.

    BaseView enables easily adding DOM Events by doing the book keeping and allowing for event delegation.

    Params:

    NameTypeAttributeDescription
    el Element

    The Element this view is responsible for. Saved to this.el, a View is always responsible For a single root element. The view can then split up it's contained elements with subviews
    options Object

    An object of options for the view
    options.breakpoints Array

    An array of breakpoint objects the view uses. If this options is specified, the breakpointChanged function will only be called with breakpoints specified here, otherwise it will be called for every breakpoint change.

    public addDomEvent(type: String, listener: function, selector: String | Element) source

    Bind a DOMEvent to the view, optionally filtered on the selector.

    Params:

    NameTypeAttributeDescription
    type String

    Event typoe
    listener function

    Event listener function that will be scoped to this view
    selector String | Element
    • optional
    • default: this.el

    If selector is a String, the string will be used to test matching using delegated events to this.el. If it's an Element then bind event directly to that element. Binding directly to an element is usefull for events that don't bubble. (form submit, for example)
